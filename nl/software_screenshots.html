Schermafdrukken van BibleTime

New Sword setup dialog.  Since version 1.4 BibleTime offers a new dialog to manage Sword modules. It is now possible to setup Sword's configuration, install modules and uninstall modules using a userfriendly interface. Large screenshot

Verbeteringen in 1.3.  Hier ziet u enkele verbeteringen in BibleTime 1.3. De boomstructuur aan de linkerzijde is nu opgezet per moduletype en -taal. De Griekse tekst in het rechter weergavescherm wordt nu beter weergegeven, omdat BibleTime nu voor iedere taal eigen lettertype-instellingen ondersteunt. Grote schermafdruk

Nieuwe zoekdialoog.  De zoekdialoog in eerdere versies was niet makkelijk te gebruiken. Dit is de nieuwe zoekdialoog samen met rechts het nieuwe modulekiesscherm. Grote schermafdruk
