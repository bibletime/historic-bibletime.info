Installation of Sword's text modules

This page explains how to add the text modules, which are available from Crosswire, to the BibleTime software to make it really useful.

The modules are available as ZIP files from the Crosswire Bible society. Have a look into the »Modules« section of the BibleTime web pages to get an overview of the available modules.

Before you start installing modules please make sure BibleTime is installed!

First you have to find the directory, which we'll call now »Sword directory«, on your system. The Sword directory contains all Sword data, i.e. it contains at least the sub-directories »modules/« and »mods.d/«.

Then you have to unzip the downloaded ZIP file(s) into the Sword directory to populate the two subdirectories we mentioned.

After restarting BibleTime the modules should appear in the main index on the left side of the main window.

The installation instructions made by the Sword team are available here:

www.crosswire.org/sword/modules/moduleinstall.jsp.

If you get into troubles please read the answers to the frequently asked questions. The FAQ is available on the left side of this page.
