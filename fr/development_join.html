Rejoindre l'équipe BibleTime

C'est pour nous une joie de voir de nombreuses personnes nous aider à améliorer le logiciel BibleTime, ainsi que la documentation et les traductions.

La marche à suivre...

Cette démarche est nécessaire pour tous les types de membres de l'équipe. Veuillez la suivre avec attention!

  1. Si vous ne possédez pas de compte Sourceforge, veuillez en créer un. Cette procédure est gratuite. Nous avons besoin de votre nom d'utilisateur pour vous ajouter aux membres de l'équipe, pour vous confier des tâches, etc.

  2. Inscrivez-vous à la liste de diffusion de développement de BibleTime. Tous les développeurs lisent cette liste, pour que chacun soit au courant de l'actualité de l'équipe.

  3. Téléchargez la version développement de BibleTime et installez-la. Si vous avez besoin d'aide à ce niveau, posez vos questions sur la liste de diffusion ou consultez la section documentation.

Les prochaines étapes concernent les développeurs du logiciel...

  1. Travaillez les tutoriaux de Qt3 et de KDE3 pour vous habituer à la programmation utilisée pour BibleTime

  2. Si vous ne savez pas quoi faire, demandez à la liste de diffusion une tâche à accomplir. N'oubliez pas de préciser le niveau de difficulté que vous désirez.

Les prochaines étapes concernent les auteurs de la documentation...

  1. Lisez la documentation anglaise existante. Elle est située dans le répertoire docs/ du répertoire cvs de Bible Time.

  2. La documentation est stockée dans des fichiers docbook situés dans le sous-répertoire unicode de chaque partie de documentation (handbook, Bible study howto, installation, helpdialog). N'éditez pas directement les fichiers HTMLcar ils sont générés à partir des fichiers docbook. Le fichier de traduction est aussi généré à partir des données docbook.

Les prochaines étapes concernent les traducteurs...

  1. Jetez un oeil aux pages d'internationalisation du projet KDE. On peut les trouver ici : i18n.kde.org

  2. Installez le programme KBabel, s'il n'est pas encore installé sur votre système. Il sera une aide précieuse dans le processus de traduction.

  3. Lisez la page suivante : traduire BibleTime

Un instant...

Avez-vous vu "La Passion du Christ" ? Le film vous a-t-il parlé ? Peut-être ne l'avez-vous pas vu ? -- dans tous les cas, si vous voulez en savoir plus sur la passion de Jésus-Christ, vous pouvez visiter le site "the good news", qui vous donnera plus d'information à ce sujet, même en Français !

Traductions

Cela ne vous prendra que quelques minutes de mettre à jour la traduction du site dans votre langue. L'aide à la traduction est quelquechose que n'importe quel utilisateur peut faire, et qui aide vraiment tous les utilisateurs. Si vous voulez nous aider à garder les traductions à jour, veuillez lire la page comment traduire et mettez à jour le fichier de traduction dans votre propre langue.
