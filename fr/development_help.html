Aidez-nous

Notre but est de faire de BibleTime un outil d'étude de la Bible vraiment mature qui puisse être utilisé par les chrétiens pour grandir dans leur relation avec Lui! A cet effet nous avons besoin de vous! Vous pouvez nous aider de nombreuses façons. Veuillez lire cette page pour savoir comment nous aider à améliorer BibleTime.

Voulez-vous prier?

Tout le monde est capable de prier. Si vous priez pour les projets Sword et BibleTime vous nous aidez autant que les autres membres du projet BibleTime.

Savez vous développer des logiciels?

Savez-vous programmer en C++? Etes-vous familier avec l'API de KDE ou motivé à apprendre? Si vous avez du temps pour travailler régulièrement sur le code source de BibleTime veuillez rejoindre notre équipe! Pour nous rejoindre, lisez la page "Joindre l'équipe" dans cette section.

Savez-vous écrire de la documentation?

Pensez-vous que vous seriez bon pour travailler sur la documentation anglaise de BibleTime? Si oui, veuillez lire la page "Joindre l'équipe" de cette section.

Êtes-vous bon pour traduire de l'anglais vers une autre langue?

BibleTime devrait être accessible à tous. C'est pourquoi nous avons besoin d'une traduction dans votre langue, si elle n'est pas disponible. Si vous décidez de travailler sur une traduction contactez nous à info@bibletime.info pour voir si aucun autre traducteur ne l'a commencé. Nous vous fournirons les fichiers à traduire.

Veuillez utiliser le programme KDE "KBabel" pour traduire les fichier! Cet outil simplifie la traduction.
