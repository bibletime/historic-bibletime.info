바이블타임 1.1 설치에 필요한 것들

  • KDE 2: 기본 라이브러리들과 기본 패키지들이 필요합니다. KDE 2는 많은 (구버전의) 리눅스 배포판에서 제공이 되고 있습니다. 보통 기본적으로 설치가 됩니다.
  • Sword 1.5.3 바이너리 패키지로 설치하지 않았을 경우에만 필요합니다. 스워드(Sword)의 설치 명령어는 이곳에서 확인 하실 수 있습니다: 스워드(Sword) 설치

바이블타임 1.1 설치에 필요한 것들

  • KDE 3: 기본 라이브러리들과 기본 패키지들이 필요합니다. KDE 3는 많은 리눅스 배포판에서 제공이 되고 있습니다. 대부분 기본적으로 설치가 됩니다.
  • Sword 1.5.6 스워드(Sword)의 설치명령어들은 이곳에서 확인 가능합니다: 스워드(Sword) 설치
  • Curl >= 7.10. This is necessary for the remote installation features of the new Sword Setup Dialog.
