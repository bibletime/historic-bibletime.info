How to link to www.bibletime.info

Please do not copy the images because they may change their design.

Larger banner

Put the following HTML code on your pages: 


 <a href="http://www.bibletime.info/" target="_blank">
 <img src="http://www.bibletime.info/images/banner/big.jpg" width="312" height="60" border="0"/>
 </a>

This will look like this:

Smaller banner

Put the following HTML code on your pages: 


 <a href="http://www.bibletime.info/" target="_blank">
 <img src="http://www.bibletime.info/images/banner/small.jpg" width="88" height="31" border="0"/>
 </a>

This will look like this:

Text link

To insert a simple text link please use the following HTML code on your pages: 


 <a href="http://www.bibletime.info/" TARGET="_blank">
 BibleTime - Bible study software for KDE
 </a>

This will look like this: BibleTime - Bible study software for KDE
