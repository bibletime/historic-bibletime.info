Was ist die Sword CD?

Das Sword Projekt bietet eine CD an, die alle Software für Windows, Linux und Mac OS X enthält.Die CD ist wie alle anderen Angebote der Crosswire Bibelgesellschaft kostenlos erhältlich! Schauen Sie weiter unten nach wie man die CD herunterladen kann. Wenn Sie die CD nicht herunterladen können, wenn Sie zum Beispiel keine schnelle Internetverbindung haben, können Sie die CD bestellen. Wenn Sie in Europa leben schicken ihnen die BibleTime Entwickler die CD; füllen Sie bitte in diesem Falle das Formular weiter unten aus.

Was kann ich mit der CD machen?

Sie können einige wichtige Dinge mit der CD machen:

  • Studieren Sie die Bibel!  Benutzen Sie all die Bibeln, Kommentare, Lexika, Wörterbücher und Bücher um Gottes Wort zu studieren. Wir hoffen, dass die CD ihnen hilft, Gottes Wort in einer effizienteren Art und Weise zu studieren!

  • Machen Sie Kopien der CD!  Bitte kopieren Sie die CD und teilen Sie diese mit vielen anderen! Wenn Sie können, dann tun Sie dies bitte kostenlos. Wenn Sie dies jedoch nicht können, verlangen Sie bitte nur das Geld um ihre Unkosten zu decken. Ansonsten wäre es nicht möglich einige Texte auf der CD zu verteilen.

Ich möchte die Sword CD herunterladen!

Kein Problem! Bitte laden Sie die Datei mit dem letzten Datum von ftp.crosswire.org/pub/sword/iso/ herunter.

Ich möchte die Sword CD bestellen!

Wenn Sie irgendwo in Europa leben, dann schicken die BibleTime Entwickler ihnen die CD. Bitte geben Sie die benötigten Informationen im Formular unten ein und schicken Sie es ab. Die BibleTime Entwickler schicken ihnen die CD umsonst.
