Installation von BibleTime

Diese Seite erklärt, wie man BibleTime installieren sollte. Bitte folgen Sie den Schritten um möglichst schnell BibleTime zum Laufen zu bekommen.

Die zwei Wege BibleTime zu installieren

Es gibt zwei Möglichkeiten, um BibleTime auf ihrem System zu installieren:

  1. Installation mit vorkompilierten Binärpaketen, wie z.B. RPM oder DEB
  2. Übersetzen Sie BibleTime selbst auf ihrem eigenen System

Wir empfehlen die Installation mit den Binärpaketen. Es ist der einfachste Weg.

Vorkompilierte Binärpakete

Wie versuchen Pakete für die wichtigsten Linux Distributionen zur Verfügung zu stellen. Diese enthalten RedHat, Mandrake und SuSE.

Laden Sie zuerst das Binärpaket für ihre Linux-Version herunter. Zum Beispiel: Sie möchten BibleTime 1.2.2. für SuSE 8.1. In diesem Falle müssen Sie das Paket "bibletime-1.2.2-1suse8.1.i586.rpm" herunterladen. Vergessen Sie auch nicht, das Paket bibletime-i18n zu installieren, da dieses die Übersetzungen des Programms und der Dokumentation enthält.

Wie Sie sehen, enthält jeder Paketname die Linuxdistribution und deren Version, für das es erstellt wurde. Versuchen Sie bitte nicht, ein Paket auf einem anderem System zu installieren, als für das es erstellt wurde.

Laden Sie BibleTime hier herunter: BibleTime herunterladen. Vergessen Sie nicht danach einige Module zu installieren Wie man Module installiert.

Übersetzung des Quellcodes

Die Installation vom Quellcode ist schwieriger als die Installation mit Binärpaketen. Bitte befolgen Sie die nächsten Schritte, wenn Sie vom Quellcode installieren wollen:

  1. Benötigte Software installieren: BibleTimes Anforderungen
  2. Get the source code package of BibeTime: Herunterladen
  3. BibleTime übersetzen und installieren 
    
	./configure --prefix=<your KDE directory>
	make
 	make install
    Sie können ihr KDE Verzeichnis mit dem Befehl kde-config --prefix herausbekommen. Bei Problemen lesen Sie bitte das FAQ: FAQ. Bei unlösbaren Problemen wenden Sie sich bitte an <info@bibletime.info>

Vergessen Sie nicht einige Module nach der Installation herunterzuladen: Wie man Module installiert.
