Installation von BibleTime

Diese Seite erklärt, wie man BibleTime installieren sollte. Bitte folgen Sie den Schritten um möglichst schnell BibleTime zum Laufen zu bekommen.

Die zwei Wege BibleTime zu installieren

Es gibt zwei Möglichkeiten, um BibleTime auf ihrem System zu installieren:

  1. Installation mit vorkompilierten Binärpaketen, wie z.B. RPM oder DEB
  2. Übersetzen Sie BibleTime selbst auf ihrem eigenen System

Wir empfehlen die Installation mit den Binärpaketen. Es ist der einfachste Weg.

Vorkompilierte Binärpakete

Wie versuchen Pakete für die wichtigsten Linux Distributionen zur Verfügung zu stellen. Diese enthalten RedHat, Mandrake und SuSE.

First download the binary package for your Linux distribution. For example: You want to download BibleTime 1.2.2 for SuSE Linux 8.1. In this case you have to download the package »bibletime-1.2.2-1suse8.1.i586.rpm«. Please don't forgot to download the package bibletime-doc to get the documentation and the translations, which are available for BibleTime.

Wie Sie sehen, enthält jeder Paketname die Linuxdistribution und deren Version, für das es erstellt wurde. Versuchen Sie bitte nicht, ein Paket auf einem anderem System zu installieren, als für das es erstellt wurde.

You can download BibleTime here: Download BibleTime. Don't forget to install some text modules after you installed BibleTime: How to install the text modules.
