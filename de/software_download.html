Wie man BibleTime bekommt

Es gibt verschiedene RPM Pakete von BibleTime. Um zu sehen, ob sie für ihe System erstellt wurden, schauen Sie sich den Dateinamen des Paketes an. Er enthält den abgekürzten Namen und die Version des Linux Systems, für das das Paket erstellt wurde. Zum Beispiel wurde das Paket "bibletime-1.2.2-1suse8.1.i586.rpm" für SuSE 8.1 Systeme gebaut die auf einem i586 oder besser laufen.

Das "bibletime" Paket enthält sowohl die Software als auch die Dokumentation in Englisch. Übersetungsdatein von beiden können im "bibletime-i18n" Paket gefunden werden.

Sie können BibleTime auf unseren SourceForge Seiten herunterladen.
