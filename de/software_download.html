Wie man BibleTime bekommt

Es gibt verschiedene RPM Pakete von BibleTime. Um zu sehen, ob sie für ihe System erstellt wurden, schauen Sie sich den Dateinamen des Paketes an. Er enthält den abgekürzten Namen und die Version des Linux Systems, für das das Paket erstellt wurde.

So wurde z.B. das Paket »bibletime-1.2.2-1suse8.1.i586.rpm« für SuSE 8.1 Systeme erstelt, die einen i586 kompatiblen oder besseren Prozessor verwenden.

Bitte laden Sie BibleTime auf den Sourceforge Seiten von BibleTime herunter:

http://www.sf.net/projects/bibletime.

Please download the packages »bibletime« and »bibletime-doc«! Without the »bibletime-doc« package you would have no handbook, daily tips or installation instructions.
