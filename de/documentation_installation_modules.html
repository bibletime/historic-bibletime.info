Installation von Swords Text-Modulen

Diese Seite erklärt, wie man die Text-Module, die bei Crosswire verfügbar sind, zur BibleTime Software hinzufügt um diese wirklich nützlich zu machen.

Die Module sind als ZIP Dateien bei der Crosswire Bibelgesellschaft verfügbar. Schauen Sie in den Abschnitt "Module" der BibleTime Internetseiten um einen Überblick über die verfügbaren Module zu bekommen.

Bevor Sie damit beginnen, Module zu installieren, stellen Sie bitte sicher sicher, dass BibleTime installiert ist!

Zuerst müssen Sie das Verzeichnis finden, dass wir im folgenden "Sword Verzeichnis" nenn. Das Sword Verzeichnis enthält alle Sword Daten, d.h. es muss zumindest die Unterverzeichnisse "modules/" und "mods.d/" enthalten.

Anschließend müssen Sie die heruntergeladenen ZIP Dateien in das Sword Verzeichnis entpacken, um die erwähnten.zwei Unterverzeichnisse mit Daten zu füllen.

Nach einem Neustart von BibleTime sollten die Module im Hauptindex auf der linken Seite des Hauptfensters erscheinen.

Die Installations-Anleitung, die vom Sword Team erstellt wurde, ist hier verfügbar:

www.crosswire.org/sword/modules/moduleinstall.jsp.

Wenn Sie Probleme bekommen, lesen Sie bitte zuerst die Antworten auf die häufig gestellten Fragen. Das FAQ ist auf der linken Seite dieser Seite verfügbar.
