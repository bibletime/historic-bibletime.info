Wie man auf www.bibletime.info verweist

Bitte kopieren Sie nicht die Bilder, da sie das Design verändern könnten.

1. Großer Banner

Verwenden Sie den folgenden HTML Code auf ihren Seiten: 


 <a href="http://www.bibletime.info/" target="_blank">
 <img src="http://www.bibletime.info/images/banner/big.jpg" width="312" height="60" border="0"/>
 </a>

Dies sieht dann wie folgt aus:

2. Kleiner Banner

Setzen Sie den folgenden HTML Code auf ihre Seiten: 


 <a href="http://www.bibletime.info/" target="_blank">
 <img src="http://www.bibletime.info/images/banner/small.jpg" width="88" height="31" border="0"/>
 </a>

Dies sieht dann wie folgt aus:

3. Textverweis

Um einen einfachen Textverweis einzügen verwenden Sie bitte den folgenden HTML code auf ihren Seiten:: 


 <a href="http://www.bibletime.info/" TARGET="_blank">
 BibleTime - Bibelsoftware für KDE
 </a>

Dies sieht folgendermaßen aus: BibleTime - Bibelsoftware für KDE

Übersetzungen

Es kostet Sie nur einige Minuten um die Übersetzung der Internetseiten in ihrer Sprache auf den neusten Stand zu bringen. Hilfe mit den Übersetzungen ist etwas, was alle Benutzer tun können und was auch wieder allen zu Gute kommt. Wenn Sie helfen möchten, lesen Sie doch bitte die Seite über Übersetzungen und bringen sie anschließend die Datei mit der Übersetzung in ihrer Sprache auf den neusten Stand.
