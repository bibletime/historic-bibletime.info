Swords Installation

Diese Seite erklärt, wie man Sword installiert, welches von BibleTime benötigt wird.

Die Installation der Sword Bibliothek wird nur benötigt, wenn Sie BibleTime vom Quellcode aus übersetzen möchten! Wenn Sie für die Installation vorkompilierte Binärpakete benutzen, dann installieren Sie Sword bitte nicht selbst! Die Bibliothek ist dann im BibleTime Binärpaket enthalten!

Installationsvorgang

Die grundlegende Installation ist auf den Seiten der Crosswire Bible Society beschrieben:

http://www.crosswire.org/sword/software/index.jsp
