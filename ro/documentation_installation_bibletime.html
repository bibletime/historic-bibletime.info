Instalare BibleTime

Aceasta pagina explica modul in care trebuie instalat Bbletime. Te rog sa urmezi acesti pasi pentru a instala rapid BibleTime.

Cele doua modalitati de instalare BibleTime

Sunt doua posibilitati pentru a instala BibleTime pe computerul tau:

  1. Instaleaza folosind pachete binare precompilate, ca RPM sau DEB
  2. Compileaza BibleTime din surse pe computerul tau

Noi iti recomandam instalarea folosind pachetele binare. Este calea cea mai simpla.

Pachete binare precompilate

Incercam sa oferim pachete pentru cele mai importante distributii Linux. Aceasta include RedHat, Mandrake si SuSE.

First download the binary package for your Linux distribution. For example: You want to download BibleTime 1.2.2 for SuSE Linux 8.1. In this case you have to download the package "bibletime-1.2.2-1suse8.1.i586.rpm". Please don't forgot to download the package bibletime-i18n to get the documentation and the translations, which are available for BibleTime.

Dupa cum poti vedea fiecare nume de pachet contine distributia Linux si versiunea pentru care a fost creat. Te rog nu incerca sa instalezi un pachet pe un alt sistem pentru care nu a fost facut.

You can download BibleTime here: Download BibleTime. Don't forget to install some text modules after you installed BibleTime: How to install the text modules.

Compilare din codul sursa

Installation using BibleTime's source code is more difficult than using binary packages. Please follow the steps below if you decided to install BibleTime from sources:

  1. Install the required software: BibleTime's requirements
  2. Get the source code package of BibeTime: Download
  3. Compile and install BibleTime: 
    
	./configure --prefix=<your KDE directory>
	make
 	make install
    You can get your KDE directory by running the command kde-config --prefix. If you get problems please read the FAQ. If there's no solution of your problem, please contact us at <info@bibletime.info>

Don't forget to install some text modules after you installed BibleTime: How to install the text modules.
