<P>

<H3>i18n("Requirements since BibleTime 1.1")</H3>
<P>
<UL>
	<LI>
		<A HREF="http://www.kde.org/" TARGET="_blank">KDE 3</A>:  i18n("At least the libraries and the base package.")
		<br>
		i18n("KDE 3 is shipped with many Linux distributions. Very often it's installed by default.")
	</LI>
	<LI>
		<A HREF="http://www.crosswire.org/" TARGET="_blank">i18n("Sword 1.5.3 or later")</A> i18n("Only required if do not install a precompiled binary package.")
		<br>
		i18n("Installation instructions for Sword are available here: ") <A HREF="/documentation/installation_sword">i18n("Sword's installation")</A>
	</LI>
</UL>
</P>
