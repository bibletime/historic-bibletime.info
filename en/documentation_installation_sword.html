Installation of Sword

This page explains how to install the Sword library, which is required by BibleTime.

The installation of the Sword library is only required if you compile BibleTime from sources yourself! If you install using prebuilt binary packages please don't install Sword from source code. It's included in each BibleTime binary package!

Installation process

The basic installation is described on the pages of the Crosswire Bible Society:
