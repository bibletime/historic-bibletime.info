Screenshots of BibleTime

New Sword setup dialog.  Since version 1.4 BibleTime offers a new dialog to manage Sword modules. It is now possible to setup Sword's configuration, install modules and uninstall modules using a userfriendly interface. Large screenshot

Improvements in 1.3.  Here you can see some improvements of BibleTime 1.3. The tree on the left is now set up by module type and language. The greek text in the right display window is now displayed better because BibleTime supports now own font settings for each language. Large screenshot

New search dialog.  The search dialog of previous versions wasn't easy to use. This is the new search dialog together with the new module chooser window on the right. Large screenshot

Translations

It will cost you just a few minutes to update the webpage translation in your language. Help with the translations is something all users can do and which help all users. If you want to help us keeping the translations up to date, please read the page about making translations and update the translation file of your own language.
