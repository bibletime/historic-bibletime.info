Como pegar o BibleTime

Existem vários pacotes RPM disponíveis para BibleTime. Para ver se eles foram feitos para o seu sistema dê uma olhada no nome do arquivo do pacote. Eles contêm o nome abreviado e versão da distribuição Linux para o qual foram feitos.

Por exemplo o pacote »bibletime-1.2.2-1suse8.1.i586.rpm« foi feito para sistemas SuSE 8.1 rodando em um i586 ou maior.

Por favor faça o download do BibleTime das páginas do projeto BibleTime no Sourceforge:

http://www.sf.net/projects/bibletime.

Por favor faça o download dos pacotes »bibletime« e »bibletime-doc«! Sem o pacote »bibletime-doc«! você não terá o livro de mão, dicas diárias ou instruções de instalação.
