Módulos adicionais para BibleTime

Um módulo é um texto ue pode ser lido dentro do BibleTime. Existem diferentes tipos de módulos: Bíblias, comentários, léxicons, devocionais diárias e livros.

Please read the following page to see how to install the downloaded modules: Installation of text modules.

Tipos de módulos disponíveis

  1. Bíblias
  2. Comentários
  3. Léxico / Dicionários
  4. Livros
