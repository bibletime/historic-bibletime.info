Nos ajude

Para tornar isto possível nós precisamos de sua ajuda! É possível nos ajudar de muitas maneiras. Por favor leia esta página se você puder nos ajudar a melhorar o BibleTime.

Você quer orar?

Orar é algo que todos podem fazer. Se você estiver orando pelo projeto Sword e BibleTime você está nos ajudando tanto quanto outras equipes do projeto BibleTime.

Você sabe programar softwares?

Você sabe como programar em C++? Você está familiarizado com a interface de programação d KDE ou está motivado a aprendê-la? Se você tiver tempo para trabalhar regularmente no código fonte do BibleTime nós lhe pedimos para se juntar ao nosso time!

Você sabe como escrever documentação?

Se você acredita que é, por favor leia a página »Junte-se ao time« nesta seção.

Você é bom em traduzir do inglês para outros idiomas?

Se você decidir iniciar o trabalho de tradução por favor nos contacte em info@bibletime.info para ver se a tradução ainda não foi iniciada por outro membro do time. Nós providenciaremos os arquivos que deverão ser traduzidos.

Por favor use o programa para KDE chamado »KBabel« para traduzir os arquivos! É uma boa ferramenta que simplifica o processo de tradução
