O que é o CD Sword?

O projeto Sword oferece um CD que contém todos os softwares para Windows, Linux e MacOS X. O CD é distribuído livremente como todas as outras coisas que a Sociedade Bíblica Crosswire oferece para você. Veja abaixo como fazer download do CD. Se você não puder baixar o CD, por exemplo porque você não tem uma conexão rápida, você pode comprar o CD. Se você vive na Europa os desenvolvedores do BibleTime enviarão o CD para você. Nesse caso, por favor, preencha o formulário abaixo e envie-o.

O que eu posso fazer com o CD?

Você pode fazer muitas coisas importantes com o CD:

  • Estude a Bíblia!  Usar todas as bíblias, comentários, léxicons, glossários e livros para estudar a palavra de Deus! Nós esperamos, que o CD ajude você a estudar a palavra de forma mais eficiente!

  • Faça cópias do CD!  Por favor copie o Cd e compartilhe com outros! Se você puder, por favor faça isso de graça. Se você não puder, por favor cobre somente o suficiente para cobrir suas despesas, de outra forma não seria possível distribuir alguns dos textos do CD.

Eu quero copiar o CD do Sword!

Sem problemas! Por favor copie o arquivo com data mais recente de ftp.crosswire.org/pub/sword/iso/. Não esqueça que esse arquivo tem mais de 600Mb, então você precisará de uma conexão rápida para baixá-lo.

Eu quero comprar o CD do SWORD!

Se você está vivendo em algum lugar da Europa os desenvolvedores do BibleTime enviarão o CD para você. Por favor preencha a informação requerida no formulário abaixo e envie. Os desenvolvedores do BibleTime enviarão os CDs gratuitamente.
