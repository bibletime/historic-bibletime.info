Como criar um link para www.bibletime.info

Por favor não copie as imagens porque elas podem mudar o desenho

Banner maior

Coloque o seguinte código HTML em sua página: 


 <a href="http://www.bibletime.info/" target="_blank">
 <img src="http://www.bibletime.info/images/banner/big.jpg" width="312" height="60" border="0"/>
 </a>

Isso aparecerá assim:

Banner menor

Coloque o seguinte código HTML em sua página: 


 <a href="http://www.bibletime.info/" target="_blank">
 <img src="http://www.bibletime.info/images/banner/small.jpg" width="88" height="31" border="0"/>
 </a>

Isso aparecerá assim:

Link de texto

Para inserir um link apenas com texto use o seguinte código HTML em sua página: 


 <a href="http://www.bibletime.info/" TARGET="_blank">
 BibleTime - Software de Estudo Bíblico para o KDE
 </a>

Isso aparecerá assim: BibleTime - Software de Estudo Bíblico para o KDE
