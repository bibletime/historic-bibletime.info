Instalação do BibleTime

Esta página explica como o BibleTime deve ser instalado. Por favor siga estes passos para ter uma instalação do BibleTime funcionando rápido.

As duas maneira de instalar o BibleTime

Existem duas possibilidades para instalar o BibleTime no seu sistema:

  1. Instalar usando pacotes binários pré-construídos, como RPM ou DEB
  2. Compilar o BibleTime usando os códigos fonte em seu sistema

Nós recomendamos a instalação usando os pacotes binários. É a maneira mais simples.

Pacotes binários pré-construídos

Nós tentamos fornecer pacotes para as distribuições Linux mais importantes. Isto inclui RedHat, Mandrake e SuSE.

First download the binary package for your Linux distribution. For example: You want to download BibleTime 1.2.2 for SuSE Linux 8.1. In this case you have to download the package "bibletime-1.2.2-1suse8.1.i586.rpm". Please don't forgot to download the package bibletime-i18n to get the documentation and the translations, which are available for BibleTime.

Como você pode ver cada nome de pacote contêm a distribuição Linux e versão para a qual foi construído. Por favor não tente instalar um pacote em outro sistema para o qual não foi construído.

Você pode baixar o BibleTime aqui: Download BibleTime. Não esqueça de instalar algum módulo de texto depois de instalar o BibleTime: Como instalar módulos de texto.

Compilação do código fonte

A instação do Bibletime a partir do código fonte é mais difícil do que usando pacotes binários. Por favor, siga os passos abaixo se você decidir instalar BibleTime a partir do código fonte.

  1. Instalar Software requerido: Softwares requeridos pelo BibleTime
  2. Obtenha o código fonte do BibleTime: Download
  3. Compile e instale o BibleTime: 
    
	./configure --prefix=<seu diretório do KDE>
	make
 	make install
    Você pode descobrir o seu diretório do KDE executando o comando: kde-config --prefix. Se você tiver problemas, por favor, leia as Perguntas Frequentes. Se você não encontrar a solução para o seu problema, por favor, entre em contato conosco em <info@bibletime.info>

Não esqueça de instalar algum módulo de texto depois de instalar o BibleTime: Como instalar módulos de texto.
