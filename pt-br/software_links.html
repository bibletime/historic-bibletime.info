Lista de páginas cristãs importantes

Existem muitas outras páginas cristãs na grande teia mundial. Algumas são mais, outras menos importantes.

Nós coletamos uma lista de páginas que são importantes em nossa opinião.

Desde que nós não somos autores das páginas lincadas nós não somos responsáveis pelo seu conteúdo, nós explicitamente nos disassociamos do conteúdo das páginas lincadas! Isto é necessário porque as leis alemãs dizem que você é responsável pelo conteúdo de páginas lincadas a menos que você explicitamente se disassocie delas.

Páginas cristãs gerais

Bible Technologies Group.  O lar do novo formato de texto de troca comum OSIS, que esperamos será usado por muitas sociedades Bíblicas e editoras do mundo

CrossWire Bible Society.  A sociedade Bíblica Crosswire é uma das poucas organizações cristãs no mundo que doam sem custo tudo o que produzem! Crosswire suporta o projeto Sword, o projeto KJV2003 e algumas outras coisas legais.

Softwares Bíblicos para Linux

Gnomesword.  Gnomesword é uma interface gráfica para o Sword usando o kit de ferramentas Gnome. Uma aplicação realmente boa. Nao deixe de checar!

Softwares Bíblicos para Windows

BibleWorks.  BibleWorks é uma das melhores aplicações para estudo da Bíblia para Windows!Muito bom mas muito caro também. Só é útil para escolas.

BibleWorkshop.  O programa Bible Workshop é uma aplicação poderosa de estudo da Bíblia para Windows, que está disponível em inglês, alemão e francês. É possível melhorar o Bible Workshop usando bibliotecas de Texto que você pode comprar.

Logos.  Logos é uma das melhores aplicações para estudo da Bíblia para Windows.Você pode comprar muitos, muitos textos para melhorá-lo.

OnlineBible.  A Bíblia online é uma aplicação poderosa de estudo da Bíblia para Windows.

Sword for Windows.  Esta é a aplicação de estudo da Bíblia para Windows feito pelo time do Sword!Sword é suportado pela Sociedade Bíblica Crosswire.

Volksbibel.  Volksbibel 2000.2 é um software de estudo da Bíblia alemão, baseado em metacartão, para Windows, Linux e Mac. Apesar de não haver tantos textos quanto em outros softwares, ele tem algumas coisas legais como suporte a mapas básico incluído. Se você é alemão, experimente!

Softwares Bíblicos para Mac

MacSword for OS X.  MacSword é um novo software de estudo da Bíblia para o novo sistema Macintosh OS X. As telas parecem muito promissoras!

Ferramentas de desenvolvimento

SourceForge.  SourceForge nos provê as ferramentas para desenvolver o BibleTime com muitos membros do time ao mesmo tempo! Nós poupamos muito tempo e dinheiro usando os seus serviços.
