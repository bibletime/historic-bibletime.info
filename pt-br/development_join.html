Junte-se ao time do BibleTime

Nõs estamos gratos por todos que nos ajudam a melhorar o software BibleTime, sua documentação e sua tradução.

Os passos básicos..

O seguinte é necessário para cada tipo de membro do time. Por favor siga-os!

  1. Se você não tem uma conta no SourceForge, por favor inscreva-se para uma. É grátis. Nós precisamos de seu nome de usuário para adicionar você aos membros do time, para designar tarefas, etc.

  2. Inscreva-se à lista de discussão de desenvolvimento do BibleTime. Todos os desenvolvedores lêm esta lista, então todos sabem o que está acontecendo.

  3. Pegue a versão de desenvolvimento do BibleTime e instale. Se você precisar de ajuda para isto peça na lista de discussão ou procure na seção de documentação.

O próximo passo para desenvolvedores de software...

  1. Trabalhe com os tutoriais do Qt 3 e KDE 3 para saber como programar as coisas que o BibleTime usa.

  2. Se você não sabe o que fazer por favor peça na lista de discussão por algumas tarefas. Não se esqueça de adicionar o nível de dificuldade que devem ter.

Os próximos passos para autores de documentação...

  1. Leia a documentação existente em inglês. Está localizada no diretório docs/ do diretório cvs do BibleTime.

  2. A documentação é guardada em arquivos docbook nos subdiretórios unicode de cada parte (livro de mão, Estudo Bíblico HowTo, instalação, diálogo de ajuda). Por favor, não edite os arquivos HTML diretamente porque eles são gerados a partir dos documentos docbook. Os arquivos de tradução também são gerados a partir dos arquivos docbook.

O próximo passo para tradutores...

  1. Dê uma olhada nas páginas de internacionalização do projeto KDE. Eles estão localizados em: i18n.kde.org

  2. Instale o programa KBabel, se ainda não estiver instalado em seu sistema. Ele irá ajuda-lo muito no processo de tradução.

  3. Por favor, leia a página sobre a tradução do BibleTime.
