Как загрузить BibleTime

Существует несколько пакетов RPM для BibleTime. Чтобы узнать есть ли пакет для вашей системы посмотрите на название файла пакета. Название файла включает краткое название и версию дистрибутива Linux, для которого этот файл предназначен.

Так, на пример, файл "bibletime-1.2.2-1suse8.1.i586.rpm" был сделан для дистрибутива SuSE 8.1, установленного на компьютере i586 и выше.

Загрузить BibleTime вы сможете на страницах проекта BibleTime на сайте Sourceforge.

http://www.sf.net/projects/bibletime.

Загрузите пакеты "bibletime" и "bibletime-doc"! Без пакета "bibletime-doc" у вас не будет справочника пользователя, советов на каждый день и инструкции по установке.
