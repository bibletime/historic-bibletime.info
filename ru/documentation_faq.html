1. Процесс установки

1.1. Куда задевалась панель инструментов?
1.2. Куда подевались советы на каждый день и документация?!
1.1.

Куда задевалась панель инструментов?

Please run configure with the parameter --prefix=<your KDE directory>. Then run make and make install as usual. The KDE directories for the widely used Linux distributions are:

  1. RedHat: /usr
  2. SuSE: /opt/kde3
  3. Mandrake: /usr

Alternatively you can run the command »kde-config --prefix« to get your KDE directory. The commands are:

  1. ./configure --prefix=<your KDE directory>
  2. make
  3. make install

1.2.

Куда подевались советы на каждый день и документация?!

Please install the package bibletime-i18n which contains the translations of the daily tips, the handbook, the installation instructions and the Bible study howto.

2. Usage problems

2.1. А можно произвести поиск по номеру Стронга (Strong's)?
2.2. Почему вместо текста в кодировке Уникоуд (Unicode) у меня на экране сплошные знаки вопроса?
2.1.

А можно произвести поиск по номеру Стронга (Strong's)?

Такой поиск позволяют версии 1.1 и выше. Для этого включите использование номеров Стронга в окне "Опции" (Sword->Установки фильтра). Затем откройте окно поиска и введите номер <0120> или любой другой номер Стронга (На английском языке существует так называемый конкорданс Стронга (Strong's), в котором каждое слово Библии имеет свой номер для поиска всех его употреблений в оригинале и всех его вариантов перевода на английский язык - примечание переводчика).

2.2.

Почему вместо текста в кодировке Уникоуд (Unicode) у меня на экране сплошные знаки вопроса?

First make sure you have the latest version of the module which has the problems! Check www.crosswire.org/sword for this.

You have to install an unicode font like Code2000, Arial Unicode MS or Bitstream Cyberbit to display the special unicode characters. Check the section "Configuring BibleTime->Options Dialog->Fonts" in the handbook of BibleTime how to setup Unicode fonts under Linux.

3.1.

The KJV shows no Strongs numbers in the NT!

Due to copyright problems the old KJV module which contained all Strongs numbers had to be removed. The project KJV2003 was started to create a public domain text with all Strong numbers in it. At this time it's not yet finished so the numbers are not available in all verses of the NT. The status of the project is available at Crosswire.

3.2.

Где я могу вести свои заметки при изучении Библии?

Для этого нужно установить модуль "Personal", который можно найти в разделе комментариев (commentary) на сайте Crosswire.Также прочтите страничку Комментарии.

3.3.

Почему один из заветов Библии пуст?

Некоторые модули содержат текст только одного завета или Нового или Ветхого.Новые версии BibleTime не должны показывать названия книг завета, текст которого отсутствует.
